## Step-by-Step: Creating Animations

### Method 1: Canva (Recommended for Beginners)

1. **Sign up** at Canva.com (free account works)
2. **Create custom size**: 1080x1920 (TikTok/YouTube Shorts)
3. **For each ad**:
   - Add background (gradient or image)
   - Add device images (Firestick, Android Box, Smart TV)
   - Add text with animations
   - Add StreamStickPro.com branding
   - Use Canva's animation features
4. **Export as MP4** (15-30 seconds)
5. **Upload** to TikTok/YouTube

**Canva Animation Tips:**
- Use "Fade In" for text
- Use "Pan" for moving elements
- Use "Zoom" for emphasis
- Keep animations smooth and simple

### Method 2: Adobe After Effects (Professional)

1. **Create composition**: 1080x1920, 30fps
2. **Import assets**: Images, logos, device photos
3. **Animate**:
   - Position keyframes
   - Scale keyframes
   - Opacity keyframes
   - Use easing for smooth motion
4. **Add text animations**
5. **Export**: H.264, 1080x1920, 30fps

**After Effects Tips:**
- Use "Easy Ease" on keyframes
- Keep animations under 15 seconds
- Use motion blur for fast movements
- Export at high quality

### Method 3: Lottie Animations (Web-Friendly)

1. **Use**: After Effects + Bodymovin plugin
2. **Create animations** in After Effects
3. **Export as Lottie** JSON
4. **Use on website** or convert to video

## Animation Timing Guide:

### 10-Second Ad Structure:
- **0-2s**: Hook (eye-catching visual)
- **2-4s**: Main message (device/benefit)
- **4-6s**: Content showcase
- **6-8s**: Website URL appears
- **8-10s**: Call-to-action button

### Key Animation Principles:
1. **Fast start**: Grab attention in first 3 seconds
2. **Clear message**: One main point per ad
3. **Smooth motion**: No jerky movements
4. **Readable text**: Large, bold fonts
5. **Brand visibility**: StreamStickPro.com always visible

## Export Settings:

### TikTok/YouTube Shorts:
- **Format**: MP4
- **Resolution**: 1080x1920 (vertical)
- **Frame Rate**: 30fps
- **Bitrate**: 5-10 Mbps
- **Duration**: 10-15 seconds
- **File Size**: Under 50MB

### Google Ads (Video):
- **Format**: MP4
- **Resolution**: 1080x1920 or 1080x1080
- **Frame Rate**: 30fps
- **Duration**: 6-15 seconds
- **File Size**: Under 50MB
