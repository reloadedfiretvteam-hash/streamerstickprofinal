# Ad Campaign Instructions

## Overview
This package contains 30 animated ad scripts for TikTok, YouTube Shorts, and Google Ads to promote StreamStickPro.com

## Files Included:
1. **01-ad-scripts.md** - Complete animation scripts with timing, visuals, and Google Ads text
2. **02-google-ads-csv.csv** - Ready-to-import Google Ads campaign file
3. **03-tiktok-youtube-descriptions.md** - Titles and descriptions for TikTok/YouTube
4. **04-instructions.md** - This file

## How to Use:

### For TikTok & YouTube Shorts:
1. Use the scripts in `01-ad-scripts.md` to create animated videos
2. Use titles and descriptions from `03-tiktok-youtube-descriptions.md`
3. Each ad is 10 seconds (perfect for Shorts format)
4. Include StreamStickPro.com prominently in each video
5. Use the suggested hashtags

### For Google Ads:
1. Import `02-google-ads-csv.csv` into Google Ads
2. Review and customize campaigns as needed
3. Set your budget and targeting
4. Monitor performance and optimize

## Animation Production Tips:

### Recommended Tools:
- **Fiverr Animators**: Search for "3D animated shorts TikTok YouTube"
- **Canva**: For simple animated graphics
- **Adobe After Effects**: For professional animations
- **Animaker**: For DIY animations

### Key Elements to Include:
- Device visuals (Firestick, Android Box, Smart TV)
- Website URL (StreamStickPro.com)
- Clear call-to-action
- Eye-catching animations
- Brand colors/logo

### Video Specifications:
- **Duration**: 10-15 seconds (ideal for Shorts)
- **Format**: Vertical (9:16) for TikTok, Square (1:1) for YouTube
- **Resolution**: 1080x1920 (TikTok) or 1080x1080 (YouTube)
- **File Format**: MP4
- **Frame Rate**: 30fps

## Content Themes:
1. Device-specific (Firestick, Android Box, Smart TV)
2. Content categories (Sports, Movies, TV Shows)
3. Cost savings (Cut cable, Save money)
4. Easy setup (5-minute guides)
5. Premium features (HBO, Showtime, etc.)

## Target Keywords:
- Firestick setup
- Android Box guide

- Cut cable

- ONN Android Box

- **Animation Production**: $50-500 per video (Fiverr)

## Success Metrics:

- Cost per click (CPC)
