╔═══════════════════════════════════════════════════════╗
║  SET UP ENVIRONMENT VARIABLES ON LIVE SITE            ║
╚═══════════════════════════════════════════════════════╝

If login doesn't work on your live site, add these environment
variables to your hosting platform.

┌───────────────────────────────────────────────────────┐
│ FOR CLOUDFLARE PAGES:                                  │
└───────────────────────────────────────────────────────┘

STEP 1: Go to Cloudflare Dashboard
────────────────────────────────────
1. Go to: https://dash.cloudflare.com
2. Login to your account
3. Select your website project

STEP 2: Go to Settings
───────────────────────
1. Click "Settings" in left sidebar
2. Click "Environment Variables"
3. Click "Add variable"

STEP 3: Add These 3 Variables
───────────────────────────────

Variable 1:
Name: VITE_ADMIN_DEFAULT_USER
Value: admin

Variable 2:
Name: VITE_ADMIN_DEFAULT_PASSWORD
Value: admin123

Variable 3:
Name: VITE_ADMIN_DEFAULT_EMAIL
Value: reloadedfirestvteam@gmail.com

STEP 4: Save and Redeploy
───────────────────────────
1. Click "Save"
2. Go to "Deployments"
3. Click "Retry deployment" or push new code

┌───────────────────────────────────────────────────────┐
│ FOR VERCEL:                                            │
└───────────────────────────────────────────────────────┘

STEP 1: Go to Vercel Dashboard
───────────────────────────────
1. Go to: https://vercel.com/dashboard
2. Login and select your project

STEP 2: Go to Settings
───────────────────────
1. Click "Settings" tab
2. Click "Environment Variables"

STEP 3: Add Variables
──────────────────────
Click "Add New" for each:

Name: VITE_ADMIN_DEFAULT_USER
Value: admin
Environment: Production, Preview, Development

Name: VITE_ADMIN_DEFAULT_PASSWORD  
Value: admin123
Environment: Production, Preview, Development

Name: VITE_ADMIN_DEFAULT_EMAIL
Value: reloadedfirestvteam@gmail.com
Environment: Production, Preview, Development

STEP 4: Redeploy
─────────────────
1. Go to "Deployments"
2. Click "Redeploy" on latest deployment

┌───────────────────────────────────────────────────────┐
│ FOR NETLIFY:                                           │
└───────────────────────────────────────────────────────┘

STEP 1: Go to Site Settings
────────────────────────────
1. Go to: https://app.netlify.com
2. Select your site
3. Go to "Site settings"

STEP 2: Environment Variables
──────────────────────────────
1. Click "Environment variables"
2. Click "Add a variable"

STEP 3: Add Variables
──────────────────────
Add each one:
- VITE_ADMIN_DEFAULT_USER = admin
- VITE_ADMIN_DEFAULT_PASSWORD = admin123
- VITE_ADMIN_DEFAULT_EMAIL = reloadedfirestvteam@gmail.com

STEP 4: Redeploy
─────────────────
1. Go to "Deployments"
2. Click "Trigger deploy" → "Deploy site"

┌───────────────────────────────────────────────────────┐
│ IMPORTANT NOTES:                                       │
└───────────────────────────────────────────────────────┘

1. The code has HARDCODED FALLBACK credentials, so login
   should work even without environment variables!

2. Username: admin
   Password: admin123

3. After adding env vars, you MUST redeploy your site
   for them to take effect.

4. For security, change the password in production by
   updating the VITE_ADMIN_DEFAULT_PASSWORD variable.
